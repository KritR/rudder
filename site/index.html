Rudder
npm GitHub

The OS for running coding agents.

Rudder is a native terminal workspace for Claude Code and Codex. Start tasks, watch real agent processes, keep every run isolated, review changes with Hunk, and merge completed work with one key. Rudder Cloud lets runs keep going away from your laptop.

npm install -g @viraatdas/rudder@latest rudder
Three panes, one flow Agents on the left, a real PTY worker on the right, task input at the bottom.
Native agents The worker pane is the real Claude Code or Codex process, not a fake transcript.
Clean isolation Each prompt runs in its own git worktree so parallel tasks do not collide.
Rudder Cloud Log in with rudder login, open /cloud or /sail in the dashboard, list cloud workers, and onload a worktree when a local run should continue remotely.
Local to cloud Cloud onload is built around the same worktree and merge flow: prepare isolated source state, run remotely, review locally, then merge intentionally.
No-secret safety Cloud snapshots include useful config paths, while filtering AWS credentials, .env files, SSH keys, Docker auth, kube config, and private keys.
Hunk review built in Press v to open a light-mode Hunk review, then press m from the agent list to merge the completed worktree back into your branch.
