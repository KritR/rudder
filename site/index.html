Rudder
npm GitHub Sign in

The terminal OS for coding agents.

Run multiple agents the way they should be run: in parallel, isolated, reviewable, easy to merge. Start local Claude Code or Codex workers, continue them in the cloud when it makes sense, review the diff, and merge cleanly.

npm install -g @viraatdas/rudder@latest rudder
Multiple agents in parallel. Optional cloud continuation. Code diff. Backend native plan mode. One key merge. Easy start.

Focus

TabCycle agents, worker, and task panes.
Option-1/2/3Jump straight to agents, worker, or task.
Ctrl-GUse Rudder shortcuts while a worker is focused.
Ctrl-CLeave Rudder from any pane.

Agents

j / kSelect an agent in the left pane.
vOpen or close the live diff review.
m / MMerge selected worktree, or all completed worktrees.
ddDelete an agent and remove its worktree after confirmation.

Task

/modelChoose Claude or Codex, then model and effort.
/planUse native read-only planning before implementation.
/loginAuthenticate Rudder Cloud in the browser.
/cloudList cloud workers, or start one with a task.
/sailShort alias for starting a cloud worker.

Worker

EnterSend input to Claude Code or Codex when the worker is focused.
Shift-EnterInsert a new line in the worker prompt.
PageUp/DownScroll Rudder's captured worker output.
Cmd-CCopy selected worker or task text.

Review

vSwitch from the worker to Hunk diff review.
EscLeave review and return to the worker.
wheelScroll the focused pane with trackpad or mouse.
mMerge from review when the selected agent is ready.

Cloud

rudder loginOpen browser auth and connect this machine.
rudder cloud listShow cloud workers for your account.
rudder cloud onloadMove a local run into the cloud.
rudder sailStart a Fly Machines worker from the current repo.
