Rudder
npm GitHub

The terminal OS for coding agents.

Rudder opens a local dashboard where each prompt becomes a real Claude Code or Codex worker in its own git worktree. Watch the agent live, review the diff, steer it when needed, then merge the finished work back into your branch. Use Rudder Cloud when a run should keep going away from your laptop.

npm install -g @viraatdas/rudder@latest rudder

The left pane tracks every agent, the right pane is the live Claude Code or Codex process, and the bottom pane starts new tasks. Rudder keeps each task isolated in a worktree so you can run, review, and merge work intentionally.

Real worker panes Claude Code and Codex run as native terminal processes, not rendered transcripts.
Worktree isolation Every prompt gets its own checkout so parallel agents do not fight over files.
Review and merge Open a Hunk review with `v`, then merge the selected completed agent with `m`.
Designed for long runs Rudder keeps run state, lets you restart stopped agents, and is adding cloud workers for offloaded tasks.
Rudder Cloud Run `rudder login`, then use `/cloud` or `/sail` to list, start, pause, resume, or onload a worktree to a cloud worker.
