The terminal OS for coding agents.

Rudder leans into how you should use agents: massively parallel, isolated, reviewable, and easy to merge. Start local Claude Code or Codex workers, continue agents in the cloud when it makes sense, review code diffs, and merge finished work cleanly.

npm install -g @viraatdas/rudder@latest rudder
Massively parallel agents. Optional cloud continuation. Code diff. One-key merge. Easy start.
